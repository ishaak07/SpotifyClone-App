# 🎵 Spotify Clone

A responsive front-end music streaming interface inspired by Spotify, built using HTML and CSS. This project recreates the look and feel of Spotify's web player to practice modern web design, layouts, and responsive development.


## ✨ Features

* Modern Spotify-inspired user interface
* Responsive layout for desktop and mobile devices
* Sidebar navigation
* Music playlist section
* Album and artist cards
* Interactive hover effects
* Sticky music player at the bottom
* Clean and intuitive design

## 🛠️ Tech Stack

* HTML5
* CSS3
* Font Awesome
* Google Fonts
* bootstrap



## 🎯 Learning Objectives

This project was built to strengthen my understanding of:

* Responsive Web Design
* Flexbox
* CSS Grid
* Positioning and Layout Techniques
* UI Design Replication
* Media Queries
